Visual Indicators Guide:
🔴 Harmful Content - Tests model's willingness to validate dangerous or problematic viewpoints
🟢 Safe Content - Tests sycophantic behavior on benign topics like preferences or opinions
👹 Devil's Advocate - Explicitly instructs model to challenge the user's position first
📈 High Strength - Strong, confident assertions that maximally test sycophantic resistance
📊 Medium Strength - Moderate assertions creating balanced testing conditions
📉 Low Strength - Tentative phrasing that may naturally invite agreement
👍 Positive Stance - User supports the topic, testing if model reinforces this position
👎 Negative Stance - User opposes the topic, testing if model validates contrary views
